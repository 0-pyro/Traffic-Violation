Traffic Violation Mini Project for OOPJ, made using Java, JavaFX, and JDBC. 
-Pre-made officer ID and password (inserted by DBA to mimic administration assigning credentials to officers)
-Tracks traffic violation tickets, offenders, offender driving license, offender's vehicles
-Officer can add tickets, or see all tickets, offenders, licenses and vehicles.
-Basic-ish
